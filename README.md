1.  # Introdución

# 1.1 Descripción del sistema

El programa Small Solution, es un software para búsqueda y visualización de los campos de información de un Blog de la empresa Bik Solutions. El proyecto fue desarrollado en JAVA por empleados de la compañía de ingeniería RRRR.[[SAJD1]](#_msocom_1) 

1.  #  Requisitos previos

# 2.1 Conocimientos mínimos del usuario

Small Solution posé una interfaz amigable e intuitiva, esto le permitirá hacer uso de software sin tener conocimientos previos de informática avanzada. Por ende, podemos resumir los requisitos mínimos que debe tener el usuario para el correcto uso del software de la siguiente manera:

*   Saber leer y escribir
*   Manejo básico del computador
*   Capacidad de instalar y desinstalar programas informáticos
*   Conocimiento de la ubicación de los archivos del software
    1.  # Requisitos técnicos

    2.  # Capacidades técnicas mínimas del equípo

El equipo debe contar con JAVA…[[SAJD2]](#_msocom_2) 

1.  # Complementos necesarios

Para poder hacer uso del software por primera vez, el usuario debe contar con la inforación que se va a organizar. Esta información debe estar en formato Json y contenida en un archivo .txt. Los .txt deben tener un nombre específico. (Ver 4.0 Funciones básicas)
